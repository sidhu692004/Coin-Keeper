# 🪙 Coin Keeper

**Coin Keeper** एक स्मार्ट और सरल बजट ट्रैकिंग ऐप है जो आपकी दैनिक, मासिक और व्यक्तिगत या रूम/होस्टल खर्चों को मैनेज करने में मदद करता है। Firebase Authentication और SQLite का उपयोग करके डाटा को सुरक्षित और कुशलता से स्टोर किया गया है।

## 📱 Features

- 🔐 Firebase Authentication (Secure Login)
- 🧾 Add & Track Daily Expenses
- 📊 Monthly Budget Management
- 👥 Room/Hostel Budget Splitting (Multiple Members)
- 📅 View Past Months' Reports
- 🔄 Auto-reset Budget at Start of Month
- 📈 Budget vs Expense Visuals (Upcoming)
- ☁️ Firebase Realtime Database Integration (for syncing)

## 🛠️ Technologies Used

- Kotlin (Android Studio)
- Firebase (Auth + Realtime Database)
- SQLite (for local expense storage)
- MVVM Architecture (optional)
- SharedPreferences (for storing budget type)
